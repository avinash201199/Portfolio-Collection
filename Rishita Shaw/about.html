Finalist

Smart India Hackathon Finalist for DK734

Every year approx. 5000 post graduate ayush students submit MD/PHD thesis to their respective universities. Though thesis of some universities are being uploaded on shodhganga, the thesis are not detected in plagiarism software. Hence, there is no mechanism to detect plagiarism in thesis submitted in MD/PHD Thesis. Hence we made a universal database for MD/PHD thesis submitted under Ayush system of medicine which supports plagiarism detection in thesis. With user side as well as admin side. It should automatically check the plagiarism from standard plagiarism checker plugin integrated in the system once the user or admin submit the thesis in the platform and generate the plagiarism report. It only allows submission if it pass the plagiarism(Less than 5%) test otherwise platform should deny the submission.
