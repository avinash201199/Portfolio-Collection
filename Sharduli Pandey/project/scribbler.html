This is a three-page blogging website with a homepage, a list of blogs, and a specific blog page. It includes features such as sign in and sign up, create a post, where all posts are displayed in cards format with the author's name, subject line, and a portion of the blog content, delete a specific blog, view a specific blog, edit a specific blog, and like or comment on blogs, feature. HTML, CSS, JavaScript, and Bootstrap were all used to create this.
